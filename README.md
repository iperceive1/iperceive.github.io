# I-Perceive — anonymous project page

Static project page hosted at <https://iperceive1.github.io/>.

This repository contains the public, anonymous project page for the
I-Perceive paper currently under review. It bundles:

- A short paper landing section (title, anonymity disclaimer, abstract).
- An embedded online inference demo backed by the private API gate at
  `https://api.i-perceive.com`.

The page is plain static HTML/CSS/JS — no build step. Edit
`static/js/config.js` to set the paper / code URLs once the paper is
de-anonymised.
